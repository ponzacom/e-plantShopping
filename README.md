# Paradise Nursery Shopping Application

Welcome to Paradise Nursery - Where Green Meets Serenity!
(e-plantShopping)

## Project Description

Paradise Nursery is a modern e-commerce web application for browsing and purchasing houseplants. Built with React and Redux, this application provides a seamless shopping experience for plant enthusiasts.

## Features

- **Landing Page**: Beautiful welcome page with company information and call-to-action
- **Product Listing**: Browse through various categories of houseplants
- **Shopping Cart**: Add, remove, and manage plant quantities
- **Redux State Management**: Efficient state management for cart operations
- **Responsive Design**: Works seamlessly across different devices

## Plant Categories

1. Air Purifying Plants
2. Aromatic Fragrant Plants
3. Insect Repellent Plants
4. Medicinal Plants
5. Low Maintenance Plants

## Technologies Used

- React.js
- Redux Toolkit
- CSS3
- Vite

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open your browser to the local development server
